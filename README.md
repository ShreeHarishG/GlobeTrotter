# 🌍 GlobeTrotter

**Plan Trips. Visualize Journeys. Travel Smarter.**

GlobeTrotter is a modern, full-stack travel planning application designed to simplify the complexities of organizing multi-city adventures. Whether you're planning a weekend getaway or a month-long international tour, GlobeTrotter provides the tools you need to build perfect itineraries, manage budgets, and share your plans with friends.

---

## ✨ Key Features

- **🗺️ Smart Itineraries**: Build detailed, day-wise plans across multiple cities with an intuitive interface. Keep all your flight details, hotel bookings, and activities in one place.
- **💰 Budget Awareness**: Gain full visibility into your travel expenses before you even lock your door. Track estimated costs and stay within your financial goals.
- **🤝 Share & Collaborate**: Planning with friends? Easily share your itineraries or explore public trips for inspiration.
- **📱 Responsive Design**: Access your travel plans seamlessly on any device, from your desktop at home to your mobile while on the go.
- **🔒 Secure & Private**: Your data is yours. Manage your profile and keep your travel plans as private or as public as you choose.

---

## 🛠️ Tech Stack

Built with a focus on performance, scalability, and modern developer experience:

- **Frontend**: [Next.js 16 (App Router)](https://nextjs.org/)
- **Logic**: [React 19](https://react.dev/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Developer Tools**: [ESLint](https://eslint.org/), [PostCSS](https://postcss.org/)

## 🚀 Getting Started

Follow these steps to set up the project locally on your machine.

### Prerequisites

- [Node.js](https://nodejs.org/) (Version 18.x or higher recommended)
- [npm](https://www.npmjs.com/)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/YourUsername/GlobeTrotter.git
   cd GlobeTrotter
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

### Development

Run the development server to see your changes in real-time:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to explore the application.

### Building for Production

To create an optimized production build:

```bash
npm run build
npm run start
```
